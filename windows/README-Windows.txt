Second Brain AI
================

Build dependencies:
- Python 3.11+
- Node.js LTS
- Inno Setup 6
- Optional helper: run "setup_build_env.bat" before "build_windows.bat"

How to run:
1. If you used the installer, launch "Second Brain AI" from Start Menu or desktop shortcut
2. If you are using the portable folder, double-click "Start SecondBrainAI.bat"
3. Your browser should open automatically
4. If it does not, open: http://127.0.0.1:8765

Where your local data lives:
- data\second_brain.db
- data\chroma_store\
- data\sample_docs\

Tips:
- Keep the full folder together with the EXE
- If Windows Defender prompts you, allow the app to run locally
- To stop the app, close the opened app window/process
- The installer places the app under your local user profile, so admin rights are not required
