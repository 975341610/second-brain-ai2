# Changelog - v7.1.0 (✨ 飞书级表格与 Notion 式体验)

### 🚀 重磅更新：飞书级多维表格系统
- **列类型引擎**: 支持文本、数字、状态、日期、多选等多种列属性定义。
- **智能交互**: 优化了行列操作菜单，支持拖动调整列宽及快速排序/过滤。
- **数据联动**: 表格内容与笔记元数据深度整合，支持复杂的属性看板视图。

### ✍️ 编辑器深度重构 (Notion-like Experience)
- **斜杠命令 (Slash Commands)**: 输入 `/` 快速调出组件库（表格、任务、提示块、媒体等）。
- **浮动菜单 (Floating Menu)**: 基于 Tiptap 官方扩展重新实现，提供更稳定的段落级快捷操作。
- **气泡菜单 (Bubble Menu)**: 选中文字时即刻弹出格式化选项，提升编辑效率。

### 📦 媒体与存储升级
- **分片上传**: 支持超大媒体文件的分片上传，网络波动不再是阻碍。
- **自动嵌入**: 自动识别拖入或粘贴的视频、音频及 YouTube 链接并进行流式渲染。
- **向量数据库修复**: 重构了 ChromaDB 向量存储路径及初始化逻辑，解决了由于数据损坏导致的 AI 助手失联问题。

### 🐛 稳定性修复
- **后端架构纠偏**: 修复了由于 `api/routes.py` 缩进错误导致的 Uvicorn 启动崩溃。
- **API 通讯校准**: 将 AI 通讯地址校准为标准 `/v1` 协议，并优化了模型配置的数据库存取逻辑。
- **Mac 风格代码块**: 进一步打磨 React Node View 实现的代码块，修正了语法高亮失效的边缘情况。
